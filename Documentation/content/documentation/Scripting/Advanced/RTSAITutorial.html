RTS AI Tutorial

Introduction

In this document we're going to expand on what we did in The RTS Prototype: Part 2 by extending the AIPlayer and unit datablock callbacks to call into a new set of decision-making scripts. These scripts will let our units do things like react to attacks, attack enemies that approach, assist friendly units, seek out ammo and health packs, and much more.

This builds directly on script enhancements and other work done in RTS Prototype: Part 2, so if you have not completed it you had better get your rear back over there and finish that before proceeding.
